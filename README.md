Zipdiff
ZIP Diff adalah aplikasi web untuk membandingkan dua file ZIP secara visual, termasuk struktur folder dan konten file teks di dalamnya.

fitur:
1. membandingkan struktur antar dua file zip
2. membandingkan isi file yang sama di kedua ZIP
3. indikator warna untuk file yang dihapus(merah), file yang ditambahkan(hijau), dan file yang sama namun isi berbeda(kuning)
4. indikator warna pada isi file (baris yang berbeda ditandai kuning)
5. filter untuk hanya menampilkan file yang berubah
6. Ekspor hasil dalam format TXT atau JSON

cara menggunakan
1. clone projek
2. double click projek .html
3. Pilih ZIP A dan ZIP B melalui tombol "Select ZIP"
4. Klik "Compare ZIPs"
5. Klik file di panel struktur untuk melihat perbedaan baris per baris
6. Gunakan checkbox "Show changed files only" untuk menyaring hanya file yang berubah
7. Ekspor hasil jika diperlukan
